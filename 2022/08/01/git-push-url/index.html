「git-push-url」推送多个仓库

Back2022-08-01

当我们有代码推送到多个仓库：

  • 多个remote （常用的是： git remote add origin ）

  • 同一个remote下多个url （类似的是： git remote set-url origin ）

在 “set-url” 无法做到添加url，而是替换url。

这个工具就为了简单添加 url ，没错就是这么简单。

用途

推送多个仓库

用法

原理

改变了一下当前目录的 “./.git/config” 文件里面的 remote 内容，添加 url 或 删除 url。

欢迎来提 PR 和 issues.
